atb_editarea

About

The atb_editarea plugin simply takes Christophe Dolivet’s EditArea, a free javascript-based editor for source code, and uses it to enhance the textareas inside your Textpattern admin area’s Presentation tab, making it quite a bit easier to edit your site's appearance via the web interface.

Developed by Adam T. Bradley.
